# website-performance-analysis

Analyzed website performance data using SQL, Excel, and Python to evaluate user behavior and traffic trends.

Worked on key web metrics such as sessions, page views, bounce rate, conversion rate, and average session duration.

Cleaned and transformed raw web analytics data by handling missing values, duplicates, and inconsistent formats.

Performed traffic source analysis (Organic, Direct, Referral, Paid) to measure channel-wise performance.

Conducted page-level performance analysis to identify high-traffic and high-bounce pages.

Built interactive dashboards in Power BI / Tableau to track KPIs and website health.

Identified performance bottlenecks and provided data-driven recommendations to improve user engagement and conversions
